Set Operations Guide

∪ Union (A ∪ B)

All items that appear in either list A or list B (or both).

∩ Intersection (A ∩ B)

Items that appear in both list A and list B.

∖ Difference (A ∖ B)

Items that are in list A but not in list B.

|A| Cardinality

The number of elements in a set.
